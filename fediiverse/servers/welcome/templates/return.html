Installation steps

Hi ! Here are the steps to connect your 3DS to the Fediverse:

1. Prepare your 3DS

First, install Luma3DS custom firmware to your 3DS.

Next, make sure the "Enable loading external modules" and "Enable game patching" options are enabled. To enable these options, hold SELECT while starting your 3DS.

Next, install Pretendo or another Nintendo Network ID server replacement to your 3DS.

(Miiverse will not launch unless your device can connect to NNID which Nintendo shut down in April 2024, so unfortunately at this time fediiverse depends on an alternative service being installed.)

2. Install the fediiverse Setup Utility

Download the fediiverse Setup Utility .cia file, then install it to your 3DS using FBI.

Information will appear on the bottom screen once the QR code scans properly. Give it a second to configure everything. Make sure the output looks right (it should end in "Please reboot")

Reboot your device, and done! Open the Miiverse applet to start exploring fediiverse. You can now close this page.

If you encounter any problems during this process, check the fediiverse GitHub page for help.
