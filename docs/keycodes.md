# 3ds onkey event keycodes
these are the keycodes emitted by the onkeyup/down/press events when different buttons are pressed on the eds

| key         | keyCode | keyIdentifier |
|-------------|---------|---------------|
| A           | 13      | Enter         |
| B           | 66      | U+0042        |
| X           | 88      | U+0058        |
| Y           | 89      | U+0059        |
| L           | 76      | U+004C        |
| R           | 82      | U+0052        |
| ZL          | 87      | U+0057        |
| ZR          | 90      | U+005A        |
| d-pad up    | 38      | Up            |
| d-pad down  | 40      | Down          |
| d-pad left  | 37      | Left          |
| d-pad right | 39      | Right         |
